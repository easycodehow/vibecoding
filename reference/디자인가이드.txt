# UI

## 디자인컨셉트
- 따뜻한 페이퍼톤 배경(#FBF6EF) 위에 보라(포인트)·로즈(강조) 두 개의 액센트 컬러를 쓰는 라이트 테마
- 히어로는 어두운 오버레이 + 보라 계열로 강한 대비, 본문 섹션은 화이트 카드 + 그레이스케일 잉크 톤으로 차분하게 정리
- 섹션 제목 앞 컬러 바(로즈)가 페이지 전체의 시각적 랜드마크 역할

## 색상

### 브랜드 / 포인트 컬러
- 퍼플 (accent, hover, 특징 섹션 화살표): #8B5CF6
- 로즈/포인트 (섹션 제목 바, 기대효과 카드 강조선): #C2447A
- 히어로 서브텍스트: #A78BFA
- 카드 제목(h4) 톤: #2D1B4E, #2D2A28 (보라 계열 다크톤, 그레이스케일 아님)

### 베이스 컬러 (CSS 변수 :root)
- --ink (본문 기본색): #241E1A
- --ink-deep (헤더 스크롤/푸터 배경): #17120F
- --stone (카드 본문 텍스트): #4A4645
- --paper (페이지 배경): #FBF6EF
- --line (구분선): rgba(36, 30, 26, 0.12)
- 카드/테이블 테두리 (변수화 안 됨): #EDE0E5

### 정의만 되어 있고 현재 미사용
- --tangerine: #FF8315
- --ember: #D9600C
- --gold: #FDE0A0

## 그레이스케일
- 본문 텍스트: ink #241E1A / stone #4A4645
- 배경: paper #FBF6EF / 카드 화이트 #FFFFFF
- 구분선·테두리: rgba(36,30,26,0.12) / #EDE0E5
- 주의: 로즈·보더 계열 포인트 컬러는 히어로 영역에는 사용하지 않음 (히어로는 보라 계열로 통일)

## 시각요소
- 섹션 제목(h2) 좌측에 5px 두께 로즈 컬러 바
- 특징/한계점: grouped-box(흰 배경+얇은 테두리) 안에서 세로 구분선으로 나뉜 4분할 카드. 제목은 2줄 높이(min-height: 2.6em)로 고정하고 상단 정렬해 1줄/2줄 제목이 섞여도 첫 줄이 맞춰지도록 처리
- 기대효과: 카드를 개별 박스로 분리, 좌측 4px 로즈 강조선 + 은은한 box-shadow로 강조, 2행 2열 고정 그리드
- 특징 섹션 전용 컴포넌트: 비교표(.compare-table), 플로우 다이어그램(.flow-diagram, 화살표는 보라색)

## 레이아웃
- 콘텐츠 최대 너비: 1280px (--content-width), 중앙 정렬
- 헤더: 높이 4.5rem, position fixed, 스크롤 시 반투명 보라 배경(rgba(45,27,78,0.35))으로 전환
- 섹션 상하 패딩: 4rem
- 카드 그리드 종류
  - 기본: auto-fit, minmax(240px, 1fr)
  - 2열 고정형(.card-grid-2): auto-fit, minmax(320px, 1fr)
  - 기대효과 전용(.effect-grid): 2열 고정(repeat(2, 1fr))
- 폰트 크기
  - 루트(html): 18px
  - 섹션 제목(h2): 30px / 소제목(h3): 1.25rem
  - 카드 제목(h3): 22px / (h4): 24px
  - 본문(카드 p·ul, 리스트, 표): 20px로 통일

## 사용자 경험(UX) 요소
- 헤더는 스크롤 여부에 따라 배경이 투명 → 반투명 보라로 전환 (scroll-aware header)
- 섹션 스크롤 시 상단 여백 확보 (scroll-margin-top: 헤더 높이)
- 네비게이션 링크 hover 시 보라(#8B5CF6)로 색상 전환
- 참고: 이 환경은 네이티브 smooth-scroll API가 동작하지 않아, 필요 시 rAF 기반 수동 이징으로 구현해야 함
